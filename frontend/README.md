# Frontend (React)

- Develop by React and Vite
- Env: node:22

## Develop Steps

- Dev web runs at port 5173

```bash
npm install
npm run dev
```

## Build Steps

```bash
npm i
npm run bulid
```

After building, Vite will create a `dist` folder. This folder contains the static files that can be served by a web server, such as Nginx.
